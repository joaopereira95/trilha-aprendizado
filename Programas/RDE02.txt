2. Explique os protocolos http e https.
Protocolo utilizado para sistemas de informação de hipermidio, distribuidos e colaborativos. possui 2 metodos, GET e o método POST

3. Qual a função do DNS e por que a necessidade de sua existência?
O DNS é um sistema de nomes de dominio, serve para procurar o ip de um link para a troca de informações de IP para rodar diversos sites em seu computador

4. Caso não existisse o protocolo TCP/IP quais os problemas aconteceriam?
Nenhum computador se comunicariam,não existiria o controle de transmissão de dados
